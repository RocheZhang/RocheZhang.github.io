夏日遛狗担心虫多怎么预防？

2017-02-25
healthy life
  1. 地点选择很重要
  2. 出门前要做好准备
  3. 玩耍后要记得清洁
  4. 回家后要时刻观察
  5. 已经被叮咬了怎么办

夏天到了，越来越多的宠物主人希望能带宠物去草丛里撒欢，看着它们和小伙伴们跑来跑去，太惬意了~等等！草丛里充满着各种虫子你知道该怎么办吗！

地点选择很重要

尽量不要在野外的草丛逗留太长时间，野外草地虫子更多。带爱宠去玩尽量选择狗狗公园或者人工培育的草坪。

More info: 了解

出门前要做好准备

去草丛难免会有一些不速之客，我们看不到这些小虫子，不能随时驱赶，但是我们可以给狗狗架起一个保护伞——[体外驱虫剂]。体外驱虫剂的原理主要是使虫子还没叮咬狗狗之前，落在皮肤上感受到驱虫剂就会产生兴奋的激素，之后它们就会兴奋而死。体外驱虫剂应该滴在宠物头骨下三指的脖颈处，这里宠物舔不到，可以避免狗狗误食。但选择驱虫剂一定要选择正品，劣质的可能对狗狗皮肤和身体都有伤害。市面上常见的驱虫剂都是有期限的，到期了主人记得再给宠物滴一支哦。

More info: 驱虫剂种类

玩耍后要记得清洁

狗狗玩耍之后尽量要洗洗澡哦，宠物洗澡记得要使用狗狗专用[沐浴液]，他们的毛发和我们的头发皮肤都是有区别的。洗澡可以露出皮肤看到有没有被虫子叮咬，还可以清洗掉刚才在草地上打滚粘在身上的泥土。日常遛狗回家也最好给宠物擦一下脸和爪子哦，它们出门总是低着头嗅，鼻子周围难免蹭上灰尘等脏东西，爪子擦干净也就不会把外面的泥土带进家中啦。

More info: 给狗狗洗澡

回家后要时刻观察

回家后也要注意观察狗狗的表现，如果狗狗总是对身体某个部位不停抓挠，那主人就要检查一下了。

More info: 宠物微动作

已经被叮咬了怎么办

如果发现狗狗已经被叮咬了，尽量即时前往医院。如果发现是蜱虫叮咬，而且虫子还留在上面，又无法去医院必须自己处理的话，一定不要硬拽，用酒精滴在蜱虫口鼻处，让它淹死，两三分钟后，用尖头镊子轻轻夹住蜱虫靠近口器的地方，将其垂直拖出。将伤口消毒，检查是否清除干净。不要轻信一些土方法，可能会刺激到虫子更叮得深。
